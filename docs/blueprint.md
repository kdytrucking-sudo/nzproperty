# **App Name**: NZ Property Ace

## Core Features:

- PDF Data Extraction: AI tool extracts key property data from uploaded PDFs, including legal descriptions, owner details, and valuation information using Gemini API and Google Cloud Vision API for OCR.
- Google Maps Preview: Displays an interactive Google Maps preview based on the entered property address.
- Data Review and Edit: Presents extracted data in an editable form for user review and modification.
- Report Generation: Generates a final property valuation report by merging the extracted and edited data into a Google Docs template using the Google Docs API.
- Content Management: Allows users to manage and update regularly changing content (economic trends, market updates) via a dedicated interface, saving changes in Firebase.
- JSON Editor: Provides a backend interface for editing the AI extraction rules in JSON format.
- Report Export: Enables users to export the generated report as a .docx file or open it directly in Google Docs.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke professionalism and trust.
- Background color: Light blue (#E8EAF6), offering a clean and spacious feel.
- Accent color: Orange (#FF9800) for calls to action and important UI elements, providing a modern contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter' (sans-serif) for body text, ensuring a clean and readable experience.
- Professional and modern icons for navigation and data presentation.
- Spacious layout optimized for desktop, ensuring a professional and uncluttered user experience.
- Subtle transitions and animations to provide feedback and guide users through the report generation process.
